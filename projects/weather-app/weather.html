The Weather Dashboard is a responsive web app that lets you search for any city and get real-time weather updates. I built this project to practice working with APIs, managing state in React, and creating a clean, user-friendly interface.

This project dynamically fetches weather data using the OpenWeather API, allowing users to quickly check temperature, weather conditions, and switch between Fahrenheit and Celsius. Whether you're on desktop or mobile, the layout adapts for a smooth experience. If you need a quick weather check, give this app a spin!
