The Gainax Films Project is a responsive web app inspired by the Studio Ghibli Film Finder. I build this project to showcase the majority of Gainax's legendary movies, TV shows, and OVAs in a clean, easy-to-use interface.

This project dynamically fetches data and makes users effortlessly search, filter, and explore films. Whether on desktop or mobile, the layout adapts for a smooth experience. If you're a fan of classic anime or want a nice show to watch give this app a spin!
